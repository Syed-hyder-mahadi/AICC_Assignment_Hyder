Assignment: Build a Text Cleaner

Date: 21/03/2026

Description

Text data collected from sources like messages, emails, or social media often contains punctuation, mixed cases, and unnecessary words called stopwords. In this assignment, we build a text cleaning program that removes punctuation, converts text to lowercase, and removes stopwords to make the text suitable for analysis.

This type of preprocessing is commonly used in Natural Language Processing tasks.

Python Code
import string
from nltk.corpus import stopwords
import nltk

# Download stopwords (run once)
nltk.download('stopwords')

# Sample text
text = "Hello!!! This is a simple Example showing how the Text Cleaner works."

# Convert text to lowercase
text = text.lower()

# Remove punctuation
text = text.translate(str.maketrans('', '', string.punctuation))

# Remove stopwords
stop_words = set(stopwords.words('english'))
words = text.split()

clean_words = [word for word in words if word not in stop_words]
clean_text = " ".join(clean_words)

print("Original Text:")
print("Hello!!! This is a simple Example showing how the Text Cleaner works.")

print("\nCleaned Text:")
print(clean_text)
Example Output
Original Text:
Hello!!! This is a simple Example showing how the Text Cleaner works.

Cleaned Text:

Steps Performed
Lowercasing – Converts all characters to lowercase.
Removing Punctuation – Deletes punctuation marks such as ! , . ?.
Removing Stopwords – Removes common words like is, the, a, this.
Conclusion

Text cleaning improves the quality of textual data and helps machine learning models focus on meaningful words. Libraries like NLTK and Python (programming language) are commonly used to preprocess text for NLP tasks.
