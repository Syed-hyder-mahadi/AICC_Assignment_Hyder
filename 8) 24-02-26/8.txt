Description

In this assignment, a dataset is loaded and analyzed using Python. The program displays the top rows of the dataset, finds the column with the highest value, counts missing values, and extracts useful insights from the data.

Python Code
import pandas as pd

# Load dataset (example: CSV file)
data = pd.read_csv("data.csv")

# Display top rows
print("Top 5 Rows of Dataset:")
print(data.head())

# Find column with highest value
max_column = data.max(numeric_only=True).idxmax()
max_value = data[max_column].max()

print("\nColumn with Highest Value:")
print(max_column, "with value:", max_value)

# Count missing values
print("\nMissing Values in Each Column:")
print(data.isnull().sum())
