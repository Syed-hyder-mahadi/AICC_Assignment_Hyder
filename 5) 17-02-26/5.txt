Description

This program prints numbers from 1 to 50 using Fizz/Buzz logic.

If the number is divisible by 3, print Fizz.
If the number is divisible by 5, print Buzz.
If divisible by both 3 and 5, print FizzBuzz.

The program also counts how many times Fizz, Buzz, and FizzBuzz appear using loops and functions.

Python Code
# Function to run FizzBuzz and count occurrences
def fizzbuzz():
    fizz_count = 0
    buzz_count = 0
    fizzbuzz_count = 0

    for num in range(1, 51):

        if num % 3 == 0 and num % 5 == 0:
            print("FizzBuzz")
            fizzbuzz_count += 1

        elif num % 3 == 0:
            print("Fizz")
            fizz_count += 1

        elif num % 5 == 0:
            print("Buzz")
            buzz_count += 1

        else:
            print(num)

    # Print counts
    print("\nCounts:")
    print("Fizz:", fizz_count)
    print("Buzz:", buzz_count)
    print("FizzBuzz:", fizzbuzz_count)

# Call the function
fizzbuzz()
