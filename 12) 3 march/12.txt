Description

In this assignment, we create a simple dataset that shows the relationship between study hours and exam marks. This helps understand how features and labels are used in machine learning models to make predictions.

Features and Labels

Feature (Input):

Study Hours (number of hours a student studies)

Label (Output):

Marks obtained in the exam

In machine learning, the feature is the input data, and the label is the value we want to predict.

Relationship Prediction

From the dataset, we observe a positive relationship between study hours and marks.
This means that as study hours increase, marks also tend to increase.

For example:

Students studying 1–3 hours score around 35–50 marks.
Students studying 7–10 hours score around 78–95 marks.

This pattern can be used by a machine learning model to predict marks based on study hours.

Conclusion

Creating datasets is the first step in machine learning. By identifying features and labels, we can train models to predict outcomes. Tools like Python (programming language) and Pandas (Python library) are commonly used to build and analyze datasets for ML projects.
